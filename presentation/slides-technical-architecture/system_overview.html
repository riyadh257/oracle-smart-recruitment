System Overview

Oracle Smart Recruitment is an AI-powered B2B SaaS platform designed specifically for the Saudi Arabian recruitment market. The system leverages advanced machine learning algorithms to match candidates with job opportunities using 10,000+ attributes, ensuring precision matching beyond traditional keyword-based approaches.

Built on a modern technology stack with React 19, tRPC 11, and Express 4, the platform delivers type-safe, real-time recruitment operations with enterprise-grade scalability and performance.

Core Capabilities
AI Matching Engine
10,000+ attribute analysis with skill, culture, and wellbeing scoring
Pay-for-Performance Billing
Automated billing based on qualified applications and scheduled interviews
Saudization Compliance
Real-time Nitaqat tracking with MHRSD and Qiwa integration
ATS Integrations
Seamless sync with existing applicant tracking systems
