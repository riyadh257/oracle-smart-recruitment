Video Interview Scheduling System

STEP 1
Request
Employer initiates interview request for qualified application
→
STEP 2
Schedule
Calendly API creates event with available time slots
→
STEP 3
Confirm
Meeting URL generated and stored in videoInterviews table
→
STEP 4
Remind
Automated reminders sent 24h and 1h before interview
→
STEP 5
Complete
Status updated, notes captured, billing triggered
Calendly Integration
• Store calendlyEventId for bidirectional sync
• Automatic meetingUrl generation with unique links
• Webhook support for real-time cancellation handling
• Configurable duration (default 30 minutes)
Status Tracking
• pending initial
• scheduled confirmed
• completed finished
• cancelled aborted
Reminder System
• Scheduled job checks scheduledTime daily
• Email notifications with meeting URL and calendar invite
• Flag reminderSent prevents duplicate sends
• Customizable reminder timing per employer preferences
Data Management
• Links to applicationId, candidateId, employerId, jobId
• Cascade delete on application removal for data integrity
• Interview notes field for post-interview feedback
• Indexed queries for performance on high-volume scheduling
