# هيكل العرض التقديمي - نظام Oracle للتوظيف الذكي

## المعلومات الأساسية
- **اسم المشروع:** Oracle Smart Recruitment System
- **اللغة:** العربية
- **عدد الشرائح:** 12 شريحة
- **الجمهور المستهدف:** المديرين التنفيذيين، مديري الموارد البشرية، أصحاب القرار

---

## هيكل الشرائح

### 1. الشريحة الافتتاحية
**العنوان:** نظام Oracle للتوظيف الذكي
**المحتوى:**
- العنوان الرئيسي بالعربية
- الشعار
- نبذة مختصرة: منصة شاملة مدعومة بالذكاء الاصطناعي لإدارة عمليات التوظيف

### 2. نظرة عامة على المشروع
**العنوان:** نظرة عامة على النظام
**المحتوى:**
- تعريف النظام
- الهدف الرئيسي: تحويل دورة التوظيف الكاملة
- التركيز الجغرافي: منطقة الشرق الأوسط (السعودية بشكل خاص)
- الميزة التنافسية: الذكاء الاصطناعي + الامتثال المحلي

### 3. التحديات والحلول
**العنوان:** التحديات التي يعالجها النظام
**المحتوى:**
- **التحديات الحالية:**
  - عدم كفاءة عمليات التوظيف (30-40% من الوقت مهدر)
  - طول مدة التوظيف
  - صعوبة الامتثال لأنظمة السعودة
  - نقص البيانات لاتخاذ قرارات مستنيرة

- **الحلول المقدمة:**
  - أتمتة العمليات المتكررة
  - تحليل السير الذاتية بالذكاء الاصطناعي
  - تتبع الامتثال التلقائي
  - تحليلات شاملة لاتخاذ القرارات

### 4. الأهداف الاستراتيجية
**العنوان:** الأهداف الاستراتيجية للنظام
**المحتوى:**
- **تحسين الكفاءة:** تقليل وقت التوظيف بنسبة 40%
- **خفض التكاليف:** تقليل تكلفة التوظيف بنسبة 30%
- **ضمان الامتثال:** تتبع تلقائي لمتطلبات السعودة والتأمينات
- **قرارات مبنية على البيانات:** تحليلات شاملة وذكاء اصطناعي

### 5. المستخدمون والأدوار
**العنوان:** أصحاب المصلحة والأدوار
**المحتوى:**
- **مدير النظام:** إدارة كاملة للنظام
- **مدير الموارد البشرية:** الإشراف الاستراتيجي والامتثال
- **المُوظِّف:** العمليات اليومية للتوظيف
- **مدير التوظيف:** مراجعة المرشحين واتخاذ القرارات
- **المُقابِل:** إجراء المقابلات وتقييم المرشحين
- **المرشح:** التقديم وتتبع الطلبات

### 6. الميزات الأساسية - إدارة المرشحين
**العنوان:** إدارة المرشحين المتقدمة
**المحتوى:**
- تحليل السير الذاتية بالذكاء الاصطناعي
- استخراج تلقائي للمهارات والخبرات
- إضافة جماعية من ملفات CSV/Excel
- امتداد Chrome لالتقاط الملفات الشخصية من LinkedIn وBayt
- بحث متقدم مع مرشحات ذكية
- مطابقة المرشحين بالوظائف بنسب مئوية

### 7. الميزات الأساسية - إدارة الوظائف والطلبات
**العنوان:** إدارة الوظائف وتتبع الطلبات
**المحتوى:**
- **إدارة الوظائف:**
  - إنشاء الوظائف بسير عمل الموافقات
  - نشر متعدد القنوات (LinkedIn, Indeed, Bayt)
  - أسئلة الفرز مع منطق الاستبعاد
  - قوالب للوظائف الشائعة

- **تتبع الطلبات:**
  - خط أنابيب مرئي بالسحب والإفلات
  - فرز تلقائي حسب المتطلبات
  - إشعارات تلقائية للمرشحين
  - إجراءات جماعية للكفاءة

### 8. الميزات الأساسية - المقابلات والتقييمات
**العنوان:** المقابلات والتقييمات الذكية
**المحتوى:**
- **إدارة المقابلات:**
  - جدولة ذكية مع اقتراحات الأوقات بالذكاء الاصطناعي
  - تكامل مع Google Calendar وOutlook
  - تذكيرات تلقائية عبر البريد والرسائل
  - نماذج تقييم منظمة

- **التقييمات:**
  - اختبارات برمجية مع تقييم تلقائي
  - مكتبة تقييمات حسب الدور والمهارة
  - تتبع النتائج ومقارنة المرشحين

### 9. الامتثال والتحليلات
**العنوان:** الامتثال التنظيمي والتحليلات الشاملة
**المحتوى:**
- **الامتثال:**
  - تتبع نسب السعودة بشكل تلقائي
  - تنبيهات عند الاقتراب من الحدود
  - تكامل مع قوى والتأمينات الاجتماعية
  - تقارير جاهزة للجهات التنظيمية

- **التحليلات:**
  - لوحات معلومات شاملة
  - مقاييس الوقت والتكلفة
  - تحليل فعالية المصادر
  - تحليلات التنوع
  - توقعات مستقبلية

### 10. البنية التقنية
**العنوان:** البنية التقنية المتقدمة
**المحتوى:**
- **الواجهة الأمامية:**
  - React 19 + Tailwind CSS 4
  - واجهات سريعة الاستجابة
  - تجربة مستخدم حديثة

- **الخادم:**
  - Node.js 22 + Express 4
  - tRPC 11 للأمان من النوع
  - معالجة فعالة للطلبات

- **قاعدة البيانات:**
  - MySQL 8.0 / TiDB 6.0
  - 157 جدول منظم
  - فهرسة شاملة للأداء

- **التخزين والذكاء الاصطناعي:**
  - AWS S3 للملفات
  - Redis للتخزين المؤقت
  - خدمات الذكاء الاصطناعي المتقدمة

### 11. خارطة الطريق المستقبلية
**العنوان:** خارطة الطريق والتطوير المستقبلي
**المحتوى:**
- **المرحلة 5 (الربع الرابع 2025):**
  - تحليل الفيديو بالذكاء الاصطناعي
  - التقييمات النفسية

- **المرحلة 6 (الربع الأول-الثاني 2026):**
  - وكلاء ذكاء اصطناعي مستقلون
  - قاعدة بيانات المواهب الإقليمية

- **المرحلة 7 (الربع الثالث-الرابع 2026):**
  - حلول العلامة البيضاء
  - دعم متعدد المستأجرين
  - تطبيقات الجوال

- **المرحلة 8+ (2027+):**
  - تقنية البلوكشين
  - الواقع المعزز
  - التوسع العالمي

### 12. الشريحة الختامية
**العنوان:** شكراً لكم
**المحتوى:**
- رسالة شكر
- معلومات التواصل
- دعوة للأسئلة
