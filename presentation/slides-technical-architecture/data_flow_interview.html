Data Flow: Interview Scheduling
1
Recruiter Initiates Scheduling
Recruiter selects candidate from application pipeline and initiates interview scheduling process through the dashboard interface.
Frontend: React + tRPC
Endpoint: interviews.initiate
Input: applicationId, candidateId
2
Smart Scheduling Queries Calendars
System queries Google Calendar and Outlook APIs to fetch real-time availability for all assigned interviewers and candidate preferences.
OAuth 2.0 Authentication
Calendar API: Google + Outlook
15-minute sync intervals
Conflict detection enabled
3
AI Generates Time Slots with Confidence Scores
Machine learning algorithm analyzes historical patterns, timezone preferences, and availability windows to generate optimal time slot suggestions ranked by confidence score (0-100).
ML Pattern Recognition
Confidence Scoring: 0-100
Timezone Optimization
Historical Data Analysis
4
Confirmation Creates Events & Sends Invitations
Upon confirmation, system creates calendar events with video meeting links, updates database records, and sends automated email invitations to all participants with calendar attachments.
Calendar Event Creation
Meeting Link Generation
Email Service: Manus + SMTP
ICS Calendar Attachments
5
Automated Reminders Scheduled
System schedules multi-channel reminders (email, SMS, push notifications) at strategic intervals: 24 hours before, 1 hour before, and 15 minutes before interview time to minimize no-shows.
Multi-channel: Email + SMS + Push
Reminder Schedule: -24h, -1h, -15m
FCM for Mobile Notifications
